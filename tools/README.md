# tools

使用できるツールリスト

## プログラムとサブコマンドの対応

|   サブコマンド   |    プログラム     |                 機能                 |
| ---------------- | ----------------- | ------------------------------------ |
| count_resolution | count_img_size.py | 画像群の解像度のヒストグラムを調べる |
| makeMP4          | cvt_img2mp4.py    | 画像群から動画(mp4)を作成する        |
| -                | png2jpg.py        | png画像群をjpg画像群に変換           |
| resize           | resize.py         | 画像群をまとめてリサイズする         |
